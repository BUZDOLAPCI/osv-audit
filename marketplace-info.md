# Dedalus Marketplace Info

## Short Description
Scan dependencies for vulnerabilities using OSV database

## Long Description
Parse lockfiles (npm, yarn, pnpm, pip, poetry, cargo, go.mod), query the OSV vulnerability database, and get fix suggestions with safe version ranges. Tools: parse_dependencies, osv_query, suggest_fixes.

- DevOps
